Operator app

Policy Center edits roots safely. Trace views show tool calls, file reads/writes, cloud downloads, logs, diffs, and cancellable Codex tasks.

Safety defaults

Public URLs require OAuth or a private tunnel. Normal mode avoids raw shell and guides ChatGPT toward project.bundle, test.run, git.diff, and Codex Runner.
